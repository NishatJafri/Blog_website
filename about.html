About The Project

A beginner-focused blog built around clarity, curiosity, and visual storytelling.

This project shares core data science ideas in a way that feels practical, polished, and welcoming.

About this website

This blog was created to make data science feel less intimidating and more inspiring. It introduces data analytics, machine learning, data cleaning, and visualization through short, simple pages that are easy to explore.

Project objective

Build a multi-page learning site that combines web design, blog structure, and simple analytics behavior tracking.

Technologies used

HTML, CSS, JavaScript, visual storytelling principles, and dashboard-oriented thinking from tools like Power BI.

Who it is for

Students, beginners, and curious readers who want a strong first step into the world of data-driven thinking.

About me

I am a Data Analytics student who enjoys learning tools that turn information into useful decisions. This project reflects my interest in both technical topics and creative presentation, showing that educational websites can be informative and visually engaging at the same time.
